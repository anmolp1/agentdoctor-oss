# Integration Guides

This directory contains integration guides for using AgentDoctor with popular observability and agent development platforms.

## Available Integrations

- [LangSmith](./langsmith.md) - Export and analyze LangChain agent logs from LangSmith

## Coming Soon

- **Weights & Biases (W&B)** - Track agent health metrics alongside ML experiments
- **Datadog** - Stream agent health scores to your monitoring dashboard
- **GitHub Actions** - Automated PR health checks for agent code changes

## Contributing

We welcome integration guides for other platforms! See the [Contributing Guide](../../CONTRIBUTING.md#wanted-list) for more information.

Each integration guide should include:

1. Prerequisites
2. Export/setup instructions
3. Analysis examples
4. Continuous monitoring setup
5. Troubleshooting tips
6. Best practices
