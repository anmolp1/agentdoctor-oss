# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | Yes                |

## Reporting a Vulnerability

If you discover a security vulnerability in AgentDoctor, please report it
responsibly.

**Email:** security@mldeep.systems

Please include:

- A description of the vulnerability
- Steps to reproduce the issue
- The potential impact
- Any suggested fixes (optional)

## Response Timeline

- **Acknowledgment:** Within 48 hours of report
- **Assessment:** Within 7 days
- **Fix release:** As soon as practical after assessment

## Scope

This security policy covers the AgentDoctor library and CLI code itself.
It does **not** cover the AI agents or LLM services whose logs are being
analyzed. AgentDoctor is a diagnostic tool that processes log files locally
and makes no external network requests.

## Disclosure

We follow coordinated disclosure. We will work with reporters to agree on
a disclosure timeline before any public announcement.
